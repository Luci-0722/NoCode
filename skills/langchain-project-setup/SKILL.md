# LangChain 项目搭建技能

## 类别
development

## 描述
这是一个专门用于快速搭建LangChain项目的技能，包括项目结构设计、核心组件配置、最佳实践指导等。

## 关键词
langchain, project-setup, development, ai, llm, agent

## 使用场景
- 新建LangChain项目时快速搭建基础结构
- 配置LangChain核心组件和工具
- 设置项目依赖和环境配置
- 实现最佳实践和代码模板

## 功能特性
- 自动生成项目目录结构
- 配置LangChain核心组件
- 设置依赖文件和配置模板
- 提供项目初始化脚本

## 输出格式
返回完整的项目结构配置文件和实现代码，包括：
- 项目目录结构
- 核心配置文件
- 实现代码模板
- 环境配置文件

## 注意事项
- 适用于LangChain 0.2.x版本
- 支持多种AI模型配置
- 包含完整的错误处理机制
