# LangSmith追踪分析技能

## 类别
analysis

## 描述
这是一个专门用于分析和优化LangSmith追踪数据的技能，包括性能分析、错误诊断、优化建议等。

## 关键词
langsmith, trace, analysis, optimization, debugging, performance

## 使用场景
- 分析LangChain代理执行性能
- 诊断运行时错误
- 优化代理配置
- 监控代理行为
- 生成分析报告

## 功能特性
- 追踪数据解析
- 性能指标计算
- 错误模式识别
- 优化建议生成
- 可视化报告生成

## 输出格式
返回详细的分析报告，包括：
- 性能指标统计
- 错误分析结果
- 优化建议列表
- 可视化图表数据
- 配置优化方案

## 注意事项
- 需要LangSmith API访问权限
- 支持多种追踪数据格式
- 包含完整的分析算法
- 提供可操作的优化建议
